# SGI_cancer_recurrence_NIMO
## Requirements
- python (2.7)
- pandas (v 0.24.2)
- matplotlib (v 2.0.0)
- numpy (v 1.16.6)
- scipy (v 1.2.2)

## Installation
All python packages can be installed via pip (https://pypi.org/project/pip/).

## Description
Codes to identify NIMO signature genes by using transcriptome- and methylome-based cell type markers.
